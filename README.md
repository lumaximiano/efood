# efood - Simulador de Pedidos

Projeto desenvolvido como parte do curso Full Stack da EBAC. Um simulador de pedidos de comida com fluxo completo de compra.

## 🚀 Tecnologias

- React com TypeScript
- Redux Toolkit (gerenciamento de estado)
- Styled Components (estilização)
- React Router DOM (navegação)

## 📦 Funcionalidades

- Listagem de restaurantes
- Cardápio por restaurante
- Adicionar/remover itens do carrinho
- Fluxo de entrega e pagamento
- Confirmação de pedido

## 🔧 Como executar

```bash
npm install
npm start
